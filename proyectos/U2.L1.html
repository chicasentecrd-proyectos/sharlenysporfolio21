Unidad 2: CSS

¿Qué es CSS?

Es el lenguaje que aplica estilo a los elementos html

¿Por qué usamos CSS?

Para separar contenido de estilo

¿Dónde se pone el CSS?

Adentro de style

¿Qué etiquetas de estilo ya hemos aprendido?Define qué cambia cada propiedad.

  • font-family:Cambia la fuente
  • background:Le da color al fondo
  • color:Le da color al texto
  • font-size:Es para ponerle tamaño a las letras
  • text-align:Alineacion del texto

Escribe un ejemplo de CSS que ponga en el fondo de todas las p's de color rojo

p {
background-color: red;
}
